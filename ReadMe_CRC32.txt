                    ***************************************************************
                    **     Инструкция по работе с программой CRC32.exe(версия от 03.07.2009)    **
                    ***************************************************************
Программа CRC32.exe предназначена для подготовки контрольной суммы на каталога с данными, предназначенными для хранения
и проверки целостности данных по истечении периода хранения. 
В процессе подготовки данных для хранения создаётся служебный файл Report_CRC32.lcr, который следует поместить в корне каталога с данными,
предназначенными для хранения.
Кроме того, создаётся файл отчёта, в котором перечислена информация о количестве файлов, на которые вычислена контрольная сумма,
а также сама контрольная сумма. Эти данные должны быть занесены в сопроводительную записку к хранимым данным. 
При проверке данных после хранения создается отчёт, в котором указывается результат проверки в виде 
количества проверенных файлов, контрольной суммы, и результата проверки.
Следует сравнить данные отчёта с данными сопроводительной записки к хранимым данным.

                                    Ниже приведено руководство по работе с программой.
        -----------------------------------------------------------------------------------------------------------------

   **********************************
   *                        Режим 1:                                  *
   *     Вычисление контрольной суммы.      *
   **********************************
Для работы с программой, запустите файл CRC.exe

С помощью пункта меню "Работа"-"Выбрать каталог-источник" выберите каталог, который вы планируете подготовить
для хранения.
Далее, с помощью пункта меню "Работа"-"Выбрать каталог для отчёта" выберите каталог, в который 
будут сохранены служебные файлы:
а) бинарный файл Report_CRC32.lcr - в котором хранятся пути и контрольные суммы на каждый файл из каталога-источника.
б) текстовый файл отчёта LogFile_CRC32.txt - в который выводится текстовый отчет о Вычислении Контрольной суммы.
см. пример отчёта:
----------   LogFile_CRC32.txt    ---------------
Отчёт создан                    : 03.07.09   14:49:25
Источник                          : :\
Файлов обработано       : 11
Контрольная сумма       : 189FAD0A
----------------------------------------------
После выбора обоих каталогов:
Каталога-источника 
и Каталога для отчёта
нажмите кнопку "Вычислить Контрольную Сумму". По нажатию кнопки программа запустится на исполнение.
После окончания подсчета бинарный файл отчёта  следует поместить в корень каталога-источника. В таком виде данные можно сдавать на хранение.

Аналогично можно начать работу непосредственно с нажатия кнопки "Вычислить Контрольную Сумму". В этом случае Вам будет предложенно
последовательно выбрать каталог источника данных и каталог для отчёта.

   **********************************
   *                        Режим 2:                                  *
   *  Проверка контрольной суммы архива. *
   **********************************

Для проверки данных, полученных после хранения, убедитесь, что в корне проверяемого каталога находится файл Report_CRC32.lcr
Без этого файла программа CRC32.exe не сможет выполнить проверку данных.
С помощью пункта меню "Работа"-"Выбрать каталог-источник" выберите проверяемый каталог.
Далее, с помощью пункта меню "Работа"-"Выбрать каталог для отчёта" выберите каталог, в который 
будет сохранён служебный файл LogFile_CRC32.txt
см. пример отчёта:

            ----------   LogFile_CRC32.txt    ---------------
Отчёт создан                    : 03.07.09   15:01:10
Источник                          : :\
Файлов обработано       : 11
Контрольная сумма       : 189FAD0A
Результат                 : Проверка прошла успешно.
             ----------------------------------------------

В случае искажения хранимых данных, информация об этом отразится в отчёте:

             ----------   LogFile_CRC32.txt    ---------------
Отчёт создан                    : 03.07.09   15:04:45
Источник                          : :\
Файлов обработано       : 11
Контрольная сумма       : 754E394C
Результат                 : Контрольная сумма изменилась. Обнаружены искаженные файлы
5.doc 	Искажена контрольная сумма
1\143.txt 	Искажена контрольная сумма
            ---------------------------------------------- 
После выбора обоих каталогов:
Каталога-источника 
и Каталога для отчёта
нажмите кнопку "Проверить Контрольную Сумму". По нажатию кнопки программа запустится на исполнение.
После окончания проверки результат будет ортажен в файле отчёта, а также продублирован на экране.

Аналогично можно начать работу непосредственно с нажатия кнопки "Проверить Контрольную Сумму". В этом случае Вам будет предложенно
последовательно выбрать каталог источника данных и каталог для отчёта.
